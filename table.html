Using colspan :

Student Information
Id no. Name CGPA Department
1326 Sheikh Jubaer 3.90 CSE
1327 Ashek Kamal 3.80 CSE
1328 Sam Smith 3.25 EEE
1320 Sheikh Rayhan 3.00 EEE
1321 Jack Reacher 3.95 CSE
1323 Christine Ruse 3.75 CSE

Explanation: Here is the very first row, we have only one data which is Student information. As its a single data which requires only one coloumn. but that affect the beauty of the table. So we have used colspan = "3"; which means we have taken extra 3 coloumns for data (Student information).

Using rowspan :

Student Information
Id no. Name CGPA Department

1327 Ashek Kamal 3.80
1328 Sam Smith 3.25 EEE
1320 Sheikh Rayhan 3.00 EEE
1321 Jack Reacher 3.95 CSE
1323 Christine Ruse 3.75 CSE

Explanation: Here in first two row, we can see the department name is similar (CSE). So we have removed the table data of department name of Id no. 1327. Then we have applied rowspan in department name of Id no. 1326. Here the data (CSE) is single and the next row is empty. So we gave rowspan="2"; which means the department name CSE covered 2 rows for single data (CSE).
